# 📊 PSLP – Probability, Statistics & Linear Programming Lab

Python programs for the **Probability, Statistics & Linear Programming** lab course, covering statistical distributions, regression analysis, and optimization techniques.

## 📖 About

This repository contains Python implementations of all practicals from the PSLP lab syllabus. Topics range from basic programming and probability theory to distribution fitting, regression, and linear programming.

## 📋 Lab Syllabus

| # | Practical | File |
|---|-----------|------|
| 1 | Simple programming concepts – matrix multiplication, loops, conditionals & plotting | `Matlab.py`, `mtlab2.py` |
| 2 | Demonstration of theoretical probability limits | `2.py` |
| 3 | Plotting Normal & Exponential distributions for various parameters | `3.py` |
| 4 | Fitting Binomial distribution for given n and p | `4.py` |
| 5 | Fitting Binomial distribution after computing mean & variance | `5.py` |

## 🛠️ Tech Stack

- **Python** — 100%
- Libraries: `numpy`, `scipy`, `matplotlib`

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/MohitCh30/PSLP.git
   cd PSLP
   ```

2. **Install dependencies**
   ```bash
   pip install numpy scipy matplotlib
   ```

3. **Run any practical**
   ```bash
   python 2.py
   ```
